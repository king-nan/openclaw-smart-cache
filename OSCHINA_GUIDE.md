```
为 OpenClaw 设计的智能学习缓存层，类似"人类小脑"的缓存系统。
核心特性：
- 高速缓存：重复查询从 2-3 秒降至 0.01ms（加速 200,000 倍）
- 自动学习：识别 3 次以上的常用请求
- 智能推荐：记录并推荐最佳工具/解决方案
- 双层架构：内存缓存（LRU）+ 磁盘缓存（SQLite）
- 零依赖：仅使用 Python 标准库

已开源在 Gitee：https://gitee.com/ai-king/openclaw-smart-cache
```
